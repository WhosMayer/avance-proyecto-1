IEEE 754 — Representación de punto flotante de 64 bits
Todo número real se almacena como (−1)^s × 2^(e−1023) × 1.mantisa. Los números como 0.1 son periódicos en base 2 (0.0001100110011…) y se truncan, introduciendo error desde el primer momento. Este error se acumula en procesos iterativos.

Gráfica 8 — El error de redondeo se acumula de forma no uniforme a medida que crece el número de iteraciones.

Error de propagación: Un pequeño error inicial en θ₀ (o el error de redondeo en cada paso) se amplifica con el tiempo en sistemas no lineales como el péndulo. La naturaleza iterativa de Euler propaga y amplifica estos errores, efecto visible en la gráfica 3.
